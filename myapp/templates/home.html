Prueba Tecnica

Se debe desarrollar un sistema web, el cual permita, con un inicio de sesión administrado, acceder a un módulo de gestión de equipos y un módulo de ejecución y estadísticas. El módulo de gestión de equipos debe permitir crear, leer, actualizar y eliminar equipos. Los equipos deben tener un nombre y una dirección IPV4, además de cualquier otro campo que le parezca pertinente.

Prueba Técnica Fullstack Developer El módulo de ejecución y estadísticas debe permitir hacer una prueba de latencia (hacer ping) a cada equipo registrado en el sistema para saber si la dirección IP del equipo responde correctamente. El dato de que si el equipo responde o no a la prueba de latencia debe ser guardado para generar estadísticas y mostrar la información resultante. Adicionalmente, se debe guardar un histórico de todas las pruebas de latencia hechas por equipo.
